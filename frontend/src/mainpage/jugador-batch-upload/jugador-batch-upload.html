Carga Masiva de Jugadores NFL
Formato del Archivo

El archivo JSON debe contener un array de jugadores con los siguientes campos:

  • id: ID temporal del jugador (número)
  • nombre: Nombre completo del jugador
  • posicion: Posición (QB, RB, WR, TE, etc.)
  • equipoNFLId: ID del equipo NFL existente
  • imagenUrl: URL de la imagen (opcional)
Importante
  • ✓ La operación es todo-o-nada: si hay un error, no se crea ningún jugador
  • ✓ No pueden existir jugadores duplicados (mismo nombre en el mismo equipo)
  • ✓ Todos los equipos NFL deben existir previamente
  • ✓ El archivo debe ser menor a 10 MB
